## About
This Next.js app presents Armando Rojano’s case studies, design snapshots, and professional background. Projects include high-impact redesigns (e.g., Liverpool’s customer experiences), B2B platforms (OX Logistics), marketplaces (Yuni, Doia), and other product design engagements. Content is data-driven to keep projects, timelines, and metadata easy to update.

## Technologies Used
- Framework: Next.js 16 (App Router) with React 19 and TypeScript.
- Styling: Tailwind CSS (utility-first) plus custom animations for motion cues.
- Assets: Next/Image for optimized media delivery; video and image covers per project.
- Tooling: ESLint and Prettier for consistent formatting and linting.

## Setup and Usage
This project runs as a standard Next.js app.

1) Clone the repository:
```bash
git clone https://github.com/r0ur/portfolio-app.git
cd portfolio-app
```
2) Install dependencies (pick your package manager):
```bash
pnpm install
# or
npm install
```
3) Start the dev server:
```bash
pnpm dev
# or
npm run dev
```
Open http://localhost:3000 to view the site.

4) Build and preview production:
```bash
pnpm build && pnpm start
```
